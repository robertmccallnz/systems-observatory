This observatory watches twelve places where a system can be intervened in — from surface-level parameters down to the paradigm it arises from — using Donella Meadows' 1999 framework. Each panel maps a Meadows tier to a public Aotearoa indicator. Panels flagged ◆ anomaly show a paradigm-challenging deviation against their own 24-period baseline. Add your reading in the community annotation section — every entry is written to a public, forkable archive.

Feed sources & how mapping works

The observatory pulls headline JSON from a curated set of RSS-to-JSON proxies (RNZ, The Spinoff, StatsNZ releases). Each headline is pattern-matched against a keyword map that assigns a provisional Meadows tier — e.g. "OCR", "interest rate" → tier 12; "Treaty", "constitution" → tier 2–3; "tax reform", "employment law" → tier 5. Community members can correct or override the automatic mapping using the Add event form above. All mappings are stored in data/news-events.json in the public repo.

How submissions become the archive

Your submission is POSTed to the Gradio Space endpoint, which appends a row to the Hugging Face dataset. The dataset is public and forkable; anyone can audit, mirror, or analyse it. No email, IP, or identifier is stored — only the fields above and a UTC timestamp. This follows the Māori Data Governance Model: public, aggregated, non-personal.
